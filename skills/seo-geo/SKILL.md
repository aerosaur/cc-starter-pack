## Google Search Console

**Performance Report:**
- Monitor clicks, impressions, CTR, position

**Core Web Vitals:**
- LCP (Largest Contentful Paint): < 2.5s
- FID (First Input Delay): < 100ms
- CLS (Cumulative Layout Shift): < 0.1

## Local SEO & GEO

**Complete Profile:**
- Accurate business name
- Correct address and service area
- Phone number
- Primary/secondary categories
- Business hours

**Review Management:**

## Content Optimization

### On-Page SEO Checklist

**Title Tags:**
- Length: 50-60 characters
- Include primary keyword near start
- Include brand name at end
- Make it compelling for CTR

**Meta Descriptions:**
- Length: 150-160 characters
- Include primary keyword
- Add call-to-action
- Clear value proposition

**Header Tags:**
```
H1: Page title (one per page, includes primary keyword)
H2: Main sections (keyword variations)
H3: Subsections (supporting keywords)
```

**Content Optimization:**
- Minimum 1,000 words for competitive topics
- Keyword density: 1-2%
- Include LSI keywords naturally
- Add multimedia (images every 300 words)
- Internal links (3-5 per 1,000 words)
- External links to authoritative sources

## Technical SEO

### URL Structure
- Short and descriptive
- Hyphens for word separation
- Lowercase only
- Include target keyword when natural
- Logical hierarchy

### Canonical Tags
```html
<link rel="canonical" href="https://example.com/preferred-url" />
```

### XML Sitemaps
- Maximum 50,000 URLs per sitemap
- Include lastmod date
- Submit to Google Search Console

### Structured Data (Schema.org)

**Organization Schema:**
```json
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "Company Name",
  "url": "https://example.com",
  "logo": "https://example.com/logo.png"
}
```

**LocalBusiness Schema:**
```json
{
  "@context": "https://schema.org",
  "@type": "LocalBusiness",
  "name": "Business Name",
  "address": {
    "@type": "PostalAddress",
    "streetAddress": "123 Main St",
    "addressLocality": "City",
    "addressRegion": "ST",
    "postalCode": "12345"
  }
}
```

## Quick Reference

### Essential SEO Formulas
```
Organic CTR = (Clicks / Impressions) × 100
Bounce Rate = (Single-page Sessions / Total Sessions) × 100
Conversion Rate = (Conversions / Sessions) × 100
Keyword Density = (Keyword Count / Total Words) × 100
```

### Common SEO Status Codes
```
200 OK - Page loads successfully
301 Moved Permanently - Permanent redirect
302 Found - Temporary redirect
404 Not Found - Page doesn't exist
500 Internal Server Error - Server problem
```
